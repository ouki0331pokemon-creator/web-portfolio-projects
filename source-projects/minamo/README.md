# みなも天然水

架空の飲料メーカー「みなも水産株式会社」の販売サイトとして制作したポートフォリオ作品です。商品の安心感を伝え、比較からカート追加まで自然に進める1ページのEC体験を設計しています。

## 主な機能

- 商品追加、数量変更、削除、小計・送料・合計の自動計算
- `localStorage`によるカート内容の保持
- スライド式カートと購入デモ案内
- 商品詳細、採水地・品質管理、ご利用ガイドの複数ページ構成
- React Routerによるページ遷移と全ページ共通のカート
- モバイルファーストのレスポンシブデザイン
- キーボード操作、フォーカス表示、動きを減らす設定への対応
- 架空情報であることを明示した商品・品質・レビュー表現

## 使用技術

- React 19
- React Router
- TypeScript
- Vite
- Vitest / Testing Library
- ESLint

## 起動方法

```bash
npm install
npm run dev
```

## 確認コマンド

```bash
npm run lint
npm run test
npm run build
```

## 注意

本サイトの商品、企業、採水地、成分値、レビューはすべて架空です。実際の商品販売、決済、個人情報の送信は行いません。
