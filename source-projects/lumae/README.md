# Lumaé Botanical Hair Oil

美容サロン監修のヘアオイルを題材にした、ポートフォリオ用の1ページLPです。髪の悩みへの共感から、商品価値、成分、使い方、レビュー、保証、購入オファーまで、読み手の不安を順番に解消する構成を実装しています。

## 主な実装

- セクション間の情報設計とCTA配置
- 商品個数に応じた価格表示と購入デモ
- モバイルファーストのレスポンシブUI
- セマンティックHTMLとキーボード操作への配慮
- 架空商品・デモであることの明示

## 使用技術

- React 19
- Vite
- CSS
- lucide-react

## 起動方法

```bash
npm install
npm run dev
```

## ビルド

```bash
npm run build
```

商品、ブランド、レビュー、購入内容はすべて架空です。実際の販売・決済・個人情報送信は行いません。
