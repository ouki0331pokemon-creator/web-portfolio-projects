# 開発ソースコード

ポートフォリオに掲載している4作品の、未圧縮の開発ソースです。各作品は独立したReact/Viteプロジェクトとして起動できます。

| 作品 | 主な技術 | 実装内容 |
| --- | --- | --- |
| [Lumaé](./lumae) | React / Vite / CSS | 美容商材LP、レスポンシブUI、購入導線のデモ |
| [みなも天然水](./minamo) | React / TypeScript / React Router / Vitest | 商品ページ、カート状態管理、複数ページ、テスト |
| [KINUHA](./kinuha) | React / TypeScript / React Router / Vitest | 検索・絞り込み、カート、お気に入り、肌診断、テスト |
| [余白珈琲](./yohaku) | React / React Router / Tailwind CSS / Framer Motion | 店舗紹介、メニュー、予約・問い合わせデモ、複数ページ |

## 確認方法

確認したい作品のディレクトリへ移動し、依存関係をインストールして起動します。

```bash
npm install
npm run dev
```

公開サイトで使用する最適化済みファイルは、リポジトリ直下の`projects`に収録しています。

## 作品について

すべて自主制作・学習制作です。ブランド、店舗、商品、レビュー、予約・購入内容などは架空で、実際の販売・決済・個人情報送信は行いません。
