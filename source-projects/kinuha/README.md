# KINUHA

敏感肌向けの架空D2Cスキンケアブランド「KINUHA」のECサイトです。デザイン、情報設計、React実装、状態管理、レスポンシブ対応、アクセシビリティへの配慮を一つの作品として提示するために制作しました。

## コンセプト

> ゆらぎやすい肌に、毎日安心して続けられる植物由来のスキンケアを。

25〜34歳の働く女性を主なターゲットに、成分や使用方法を確認しながら、自分に合う商品を安心して選べる体験を設計しています。商品、レビュー、ブランド情報はすべてポートフォリオ用の架空データです。

## 主な機能

- 商品一覧、キーワード検索、カテゴリー・肌悩み・価格絞り込み、並び替え
- 商品詳細、主要成分、フリー項目、全成分、使用方法、レビュー
- カート追加、数量変更、削除、合計計算、`localStorage`保存
- 4問の肌診断と、回答に応じたケア・商品提案
- 商品のお気に入り登録と`localStorage`保存
- ブランドストーリー、FAQ、配送・返品案内、404ページ
- 375px、768px、1440pxを基準にしたレスポンシブUI
- キーボード操作、フォーカス表示、セマンティックHTML、動きを抑える設定

## 技術構成

- React 19
- TypeScript
- Vite
- React Router
- Vitest / React Testing Library
- CSS

## デザイン

アイボリー、セージグリーン、淡いピンクを基調に、自然派で親しみやすく、過度に素朴にならないD2Cブランドのトーンを目指しました。

## 起動方法

```bash
npm install
npm run dev
```

## テストとビルド

```bash
npm test
npm run build
```

## 公開

`vercel.json`を含んでいるため、Vercelへそのままデプロイできます。公開URLはデプロイ後に追記します。
